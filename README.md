# AI Company Panel

Dedicated company frontend cloned from `ai-admin`.

## Login Source
- Companies are created from `ai-admin` in the `AI Company` registration form.
- Use company email + login password from admin-created company record.

## Setup
1. `cd ai-company`
2. `npm install`
3. Copy `.env.example` to `.env` if needed

## Run
- `npm run dev`

Frontend: `http://localhost:5174`  
Backend API: `http://localhost:5000/api`
